## Requisites

- Python3.9+

## Quickstart

On Linux:

```sh
python -m venv env
source env/bin/activate
python -m pip install -r requirements.txt
python src/main.py
```

On Windows:

```ps1
python -m venv env
.\env\Scripts\activate
python -m pip install -r requirements.txt
python src\main.py
```
